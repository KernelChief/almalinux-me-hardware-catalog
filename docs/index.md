# AlmaLinux M&E Works On Database

Welcome! This site lists community-submitted hardware reports for AlmaLinux M&E workflows.

## How to contribute
1. Run the hardware survey script to generate `almalinux_me_report.json`.
2. Open a new hardware report issue in the repo and paste the JSON.
3. After approval, a report page appears under the Results section.

## Browse results
Use the site search or your browser's find (Ctrl+F) to locate hardware models quickly.
